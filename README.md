## 本書の目的

「戦術」を個々の戦闘技法だけとして扱わず、政策目的、戦略、作戦、組織、情報、兵站、指揮官の判断がつながる体系として理解するための入門書です。成人の一般読者、ミリタリー史・安全保障研究の愛好家、読書会や図上討議の参加者を主な対象とします。

本書では、米軍・NATOの公開ドクトリン、防衛省・自衛隊の公開資料、主要な戦史研究を参照し、図解と事例を用いて概念を整理します。特定の紛争当事者を利する作戦助言ではなく、歴史・制度・意思決定を分析する教育資料として執筆します。

## 図版の凡例

- 青：味方・自組織・自系統
- 赤：敵対要因・競合する意思
- 緑：目的・望ましい状態
- 灰：環境・制約・不確実性
- 実線：直接の関係
- 破線：間接的な影響またはフィードバック

図は概念を理解するための模式図であり、実在する部隊配置、正確な縮尺、個別の作戦計画を示すものではありません。
